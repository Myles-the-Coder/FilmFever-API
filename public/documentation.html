FilmFever API

This API will return movie data (description, genre, director, image URL, whether it is featured or not) about a movie by title to the user. Additional information includes a genre description by name/title and director information by name.

BUSINESS LOGIC URL HTTP METHOD REQUEST DATA FORMAT RESPONSE DATA BODY
Return a list of all movies to user /movies GET None A JSON object holding data about all the movies.
Return data about a single movie by title /movies/[title] GET None A JSON object holding data about a single movie, including description, release year, genre(s), director, poster URL.
Example:
{ "title" : "The Godfather", "releaseYear": 1972, "description": "An organized crime dynasty's aging patriarch transfers control of his clandestine empire to his reluctant son." "genre": ["crime", "Drama"], "director": "Frank Ford Coppola" }
Return data about a genre (description) by name/title /movies/[genre] GET None A short description detailing information about a specific film genre.
Return a list of all directors /directors GET None A list of all directors currently in the database.
Return data about a director (bio, birth year, death year) by name /directors/[name] GET None A JSON object holding data about a single director.
Example:
{ "name": "Frank Ford Coppola", "bio": "Schooled in low-budget filmmaking, Francis Ford Coppola (born 1939) has gone on to direct some of the most financially successful and critically acclaimed movies in U.S. cinematic history. Francis Ford Coppola, director of The Godfather and its two sequels, would be considered one of the masters of modern cinema based on those credits alone.", "birthYear": 1939, "deathYear": null }
Allow new users to register /signup POST A JSON object holding data about the user to add.
Example:
{ "userName": "John_Doe", "email": "johndoe@gmail.com", "password": "*********" } 		A JSON object holding data about the user that was added, including an ID.
Example:
{ "id": 1, "userName": "John_Doe", "email": "johndoe@gmail.com" }
Allow users to view their account info (username, email) /accounts/[id] GET None A JSON object holding user account information.
Example:
{ "id": 1, "userName": "John_Doe", "email": "johndoe@gmail.com" }
Allow users to update their user info (username) /accounts/update/[id] PUT A JSON object holding updated information provided by user.
Example:
{ "userName": "FilmFan123", "password": "*******" } 		A message indicating that the user information was successfully updated.
Allow users to add a movie to their list of favorites /accounts/favorites[id] POST A JSON object holding information about the movie added to "favorites" list.
Example:
{ "title": "Gone with the Wind" } 		A message notification indicating the movie was added to the "favorites" list.
Allow users to remove a movie from their list of favorites /account/favorites/[id] DELETE None A message indicating that the selected movie has been successfully removed from the "favorites" list.
All existing users to deactivate their account /account DELETE None A message indicating that a user email has been removed.
