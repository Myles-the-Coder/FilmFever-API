FilmFever API

This API will return movie data (description, genre, director, image URL, whether it is featured or not) about a movie by title to the user. Additional information includes a genre description by name/title and director information by name.

BUSINESS LOGIC URL HTTP METHOD REQUEST DATA FORMAT RESPONSE DATA BODY
Return a list of all movies to user /movies GET None A JSON object holding data about all the movies.
Return data about a single movie by title /movies/[title] GET None A JSON object holding data about a single movie, including description, release year, genre(s), director, poster URL.
Example:
{ "Title" : "The Godfather", "Releaseyear": 1972, "Description": "An organized crime dynasty's aging patriarch transfers control of his clandestine empire to his reluctant son." "Genre": {Genre JSON Object}, "Director": {Director JSON Object}}
Return a list of all genres /genres GET None A JSON object holding data about all available genres
Return data about a genre (description) by name/title /genres/[genre] GET None A JSON object holding data about a single genre.
Example:
{ "Name": "Drama", "Description": "Drama is a film genre that..." }
Return a list of all directors /directors GET None A list of all directors currently in the database.
Return data about a director (Name , Bio, Birth year, Deathyear) by name /directors/[name] GET None A JSON object holding data about a single director.
Example:
{ "Bame": "Frank Ford Coppola", "Bio": "Short bio on director's life", "Birthyear": 1939, "Deathyear": null}
Allow new users to register /signup POST A JSON object holding data about the user to add.
Example:
{ "userName": "John_Doe", "email": "johndoe@gmail.com", "password": "*********", "Birthday": "1988-02-23" } 		A JSON object holding data about the user that was added, including an ID.
Example:
{ "id": 1, "userName": "John_Doe", "email": "johndoe@gmail.com" }
Allow users to view their account info by Username /accounts/[Username] GET None A JSON object holding user account information.
Example:
{ "id": 1, "userName": "John_Doe", "email": "johndoe@gmail.com" }
Allow users to update their user info by Username /accounts/update/[Username] PUT A JSON object holding updated information provided by user.
Example:
{ "userName": "FilmFan123", "password": "*******" } 		A message indicating that the user information was successfully updated.
Allow users to add a movie to their list of favorites /accounts/[Username]/movies/[MovieID] POST ID of movie that is to be pushed to the "favorites" array.
Example:
$addToSet: {Favorites: req.params.MovieID} 		A message notification indicating the movie was added to the "favorites" list.
Allow users to remove a movie from their list of favorites /account/[Username]/movies/[MovieID] DELETE None A message indicating that the selected movie has been successfully removed from the "favorites" list.
All existing users to deactivate their account /account/[Username] DELETE None A message indicating that a user email has been removed.
